Getting Started

This section will cover everything you will need to know in order to get started with Paradox.

Paradox Interface

Flow themes

Paradox boasts a number of themes that you can choose from like Tron, Toy, Ghost, Blender, Simple, Ueli, Pure Dark, Coorful Dark, Industrial, Colorful Ligth, Pure Light. The steps below will guide you to finding the themes.

  1. Step 1. Go to the View option on the menu bar.
  2. Step 2. A list will appear. click on Flow Themes.
  3. Step 3. A list of themes will appear click on any one of them to choose a different theme.

System Theme

Paradox has two system themes dark mode and light mode you can work on your prefered mode by clicking on the option dark or light on the start up page.

Creating Micro Script

The micro script consists of two blocks input and output, you can follow the steps below to create your own micro script.

  1. Step 1. Right click on the input block, a menu appears click on the add parameter option, two parameter data and exec will appear choose which ever suits your script the best.
  2. Step 2. Right Click on the output block, a menu appears, click on the add return option.
  3. Step 3. Choose the block you want to use for example the + block, and connect them to the input and output blocks.
  4. Step 4. Create a Script or go to a pre-existing script, right click on the screen the block menu pops up search for the micro you created and use it in your script.

Save Project

Follow the steps below to save your Paradox projects.

  1. Step 1. Go to the File option.
  2. Step 2. A menu will appear, click on Save Project.
  3. Step 3. Enter the name of of the project you want it saved as and click on the save button.
  4. Step 4. Create a Script or go to a pre-existing script, right click on the screen the block menu pops up search for the micro you created and use it in your script.

Enable and Disable Extension

Paradox provides you with extensions to make your scripting experience smoother, follow the steps below to enable or disable them.

  1. Step 1. Go to the extension store.
  2. Step 2. Install the extension, all the blocks.
  3. Step 3. Connect the block.

Execution Mode

Paradox has two modes the default Dataflow mode and execution mode. You can follow the steps below to work in execution mode.

  1. Step 1. Go to the top right of the screen where you see the data-flow option.
  2. Step 2. Click on the drop down menu and click on exec-flow.
